## What's Built
A learning game for 2-year-olds with:
- **Number Explore** - Tap numbers 1-10, hear them spoken, see visual objects
- **Counting Game** - Count animals (1-5), pick the right answer
- **Addition Game** - Visual math problems (sums up to 5)

## Features
- Hebrew/English language toggle (flags on home screen)
- High-quality pre-recorded audio (Microsoft Edge TTS)
  - Hebrew: HilaNeural voice
  - English: AnaNeural voice
- Celebration animations with confetti
- Toddler-friendly big buttons
- Works as PWA (add to home screen on iPad/iPhone)

## Tech Stack
- Web app (HTML/CSS/JavaScript)
- Also has SwiftUI iOS source (needs Xcode to build)
- Audio files in `/docs/audio/he/` and `/docs/audio/en/`
